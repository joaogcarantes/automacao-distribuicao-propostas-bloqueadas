# Sample data

Arquivos CSV fictícios para demonstrar a estrutura esperada do projeto.

* `propostas\_atual\_exemplo.csv`: amostra baseada na tela de propostas atuais.
* `propostas\_anterior\_exemplo.csv`: amostra baseada na tela de snapshot anterior.
* `filial\_responsavel\_exemplo.csv`: base derivada da planilha de unidade/responsável, com nomes anonimizados.
* `ferias\_exemplo.csv`: base de ausências, com responsável anonimizado.
* `redistribuicao\_ferias\_exemplo.csv`: pesos de redistribuição por ausência, com nomes anonimizados.
